# 择明论坛部署指南

## 项目简介

这是一个基于 Spring Boot + Vue3 的论坛项目，主要功能包括：
- JWT + Redis 实现 Token 管理和登录态校验
- 点赞系统使用 Redis 缓存
- 基础增删改查功能
- AI 摘要生成、标签推荐等功能

## 技术栈

### 后端
- Spring Boot 3.x
- Spring Security
- Spring Data JPA
- MySQL 8.x
- Redis
- JWT

### 前端
- Vue 3
- Vite
- Element Plus

## 部署步骤

### 1. 环境准备

#### 必需软件
- JDK 17+
- MySQL 8.x
- Redis 7.x
- Node.js 18+

#### 数据库准备
1. 创建数据库：
```sql
CREATE DATABASE zheminglt CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 修改数据库配置（backend/src/main/resources/application.properties）：
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/zheminglt?useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
spring.datasource.username=你的用户名
spring.datasource.password=你的密码
```

3. 修改 Redis 配置：
```properties
spring.data.redis.database=2
spring.data.redis.host=localhost
spring.data.redis.port=6379
spring.data.redis.password=
```

### 2. 后端部署

#### 方式一：本地运行
```bash
cd backend
mvn spring-boot:run
```

#### 方式二：打包部署
```bash
cd backend
mvn clean package -DskipTests
java -jar target/ZheMingLT-1.0-SNAPSHOT.jar
```

后端服务将在 `http://localhost:8080/api` 启动

### 3. 前端部署

#### 开发环境
```bash
cd frontend/web
npm install
npm run dev
```

#### 生产环境
```bash
cd frontend/web
npm install
npm run build
```

构建后的文件在 `frontend/web/dist` 目录，可以使用 Nginx 部署

### 4. Nginx 配置示例

```nginx
server {
    listen 80;
    server_name localhost;

    # 前端静态资源
    location / {
        root /path/to/frontend/web/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api/ {
        proxy_pass http://localhost:8080/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

### 5. 验证部署

1. 访问前端页面：`http://localhost`
2. 测试登录功能
3. 测试发帖功能
4. 测试 AI 摘要生成功能

## 核心功能说明

### 1. JWT + Redis Token 管理
- Token 生成和验证：JWTUtil
- Token 黑名单：TokenBlacklistUtil
- 登录态校验：JWTInterceptor

### 2. 点赞系统缓存
- 热门帖子缓存：HotPostServiceImpl
- Redis Key 设计：
  - `hot:posts:all` - 总榜
  - `hot:posts:day:yyyy-MM-dd` - 日榜
  - `hot:posts:week:yyyy-MM-dd` - 周榜
  - `hot:posts:month:yyyy-MM` - 月榜

### 3. 基础增删改查
- 帖子管理：PostController
- 用户管理：UserController
- 评论管理：CommentController
- 分类管理：CategoryController

### 4. AI 功能
- 摘要生成：AIController.generateSummary
- 标签推荐：AIController.recommendTags
- 标题建议：AIController.suggestTitle

## 注意事项

1. **Redis 数据库**：项目使用 Redis 数据库 2，确保 Redis 已启动且数据库 2 可用
2. **AI 功能**：需要配置 DeepSeek API Key 才能使用 AI 功能
3. **文件上传**：默认支持最大 10MB 文件上传
4. **跨域**：后端已配置跨域支持，前端开发时无需额外配置

## 常见问题

### 1. 数据库连接失败
- 检查 MySQL 是否启动
- 检查数据库配置是否正确
- 检查数据库是否存在

### 2. Redis 连接失败
- 检查 Redis 是否启动
- 检查 Redis 配置是否正确

### 3. 前端无法访问后端
- 检查后端服务是否启动
- 检查前端 API 地址配置（frontend/web/.env）
- 检查跨域配置
